# Gerenciador de Funcionários

Aplicação frontend simples para cadastro e gerenciamento de funcionários.

## O que é

O **Gerenciador de Funcionários** é um projeto em HTML/CSS/JavaScript que permite adicionar, listar e remover funcionários diretamente no navegador, sem backend.

## Principais funcionalidades

- Adicionar funcionário com nome, cargo e salário
- Listar funcionários cadastrados na sessão atual
- Remover funcionários da lista

## Tecnologias

- HTML
- CSS
- JavaScript


## Observações

- Não é necessária instalação de dependências — basta abrir o HTML.
- Os dados ficam apenas na memória da página (não persistem ao recarregar).
- Validações básicas de nome, cargo e salário estão implementadas.
